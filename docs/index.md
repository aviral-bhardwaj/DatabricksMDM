# LakeMDM

**Golden records. Zero vendor lock-in. 100% Databricks-native.**

The pure Python, rules-as-data Master Data Management engine that runs natively on Delta Lake + Unity Catalog.

## Why LakeMDM?

Traditional MDM tools create expensive silos. LakeMDM delivers governed golden records directly inside your Databricks lakehouse — with full lineage, zero-copy activation, and enterprise-grade governance.

### Key Benefits

- Pure Python core (testable locally with Pandas)
- Declarative YAML domain configuration
- Pluggable matching & survivorship rules
- Full Unity Catalog governance & lineage
- Identical logic from laptop → production
- Deployed via Databricks Asset Bundles
